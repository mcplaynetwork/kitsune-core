# secrets

This directory contains secrets that are used by the application.

**Note:** This directory is not tracked by git.

## mysql.env

```plain
CFG_MYSQL_USER=
CFG_MYSQL_DATABASE=
CFG_MYSQL_PASSWORD=
```

## restic.env

```plain
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
RESTIC_REPOSITORY=
RESTIC_PASSWORD=
```
